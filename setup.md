# 🛠️ دليل الإعداد الكامل — خطوة بخطوة

> **مكتبة مدرسة سردا الأساسية المختلطة**
> المعلمة: T. Wad Refae

---

## 📋 ما ستحصلين عليه

نظام مكتبة متكامل يطابق **ملف الجرد الرسمي** للمدرسة، مع:
- بحث وعرض الكتب للطلاب
- إدارة كاملة للمعلمة (إضافة، تعديل، حذف، إعارة، إرجاع)
- استيراد ملف الجرد الـ Excel كاملاً (980 كتاب)

---

## 🟢 الخطوة 1: إنشاء Google Sheet

1. افتحي [Google Sheets](https://sheets.google.com)
2. أنشئي ملف جديد وسمّيه: **مكتبة سردا**
3. **لا تُنشئي الأعمدة يدوياً** — سيُنشئها السكريبت تلقائياً

---

## 🟢 الخطوة 2: نشر Google Apps Script

1. من Google Sheet: **ملحقات → Apps Script**
2. احذفي الكود الموجود والصقي محتوى `/backend/AppsScript.js`
3. احفظي (Ctrl + S) باسم: **مكتبة سردا**
4. اضغطي **نشر → نشر جديد**
5. اختاري النوع: **تطبيق ويب (Web App)**
6. الإعدادات:
   - **التنفيذ بصفتي**: أنا
   - **من يمكنه الوصول**: الجميع (Anyone)
7. اضغطي **نشر** وانسخي الرابط — يبدأ بـ:
   ```
   https://script.google.com/macros/s/AKfy.../exec
   ```

---

## 🟢 الخطوة 3: ربط الواجهة بالـ API

1. افتحي ملف `/frontend/app.js` (أو `library.html` إن استخدمت الملف الموحّد)
2. ابحثي عن السطر:
   ```javascript
   const API_URL = "https://script.google.com/macros/s/YOUR_SCRIPT_ID_HERE/exec";
   ```
3. استبدلي `YOUR_SCRIPT_ID_HERE` بالرابط الكامل من الخطوة السابقة

---

## 🟢 الخطوة 4: استيراد ملف الجرد الكامل

عندك خياران:

### الخيار أ — من الواجهة (موصى به):
1. افتحي الموقع
2. اضغطي 🔑 **دخول المعلم** → كلمة المرور: `Surda123surda`
3. اذهبي لتبويب **📥 استيراد ملف**
4. ارفعي ملف `library-data.xlsx` (المرفق مع المشروع)
5. اضغطي **📥 استيراد للجرد**

سيتم استيراد 980 كتاب تلقائياً على دفعات.

### الخيار ب — مباشرة في Google Sheets:
1. افتحي ملف `library-data.csv`
2. انسخي كل المحتوى
3. الصقيه في Sheet (ابتداءً من الصف 1)

---

## 🟢 الخطوة 5: نشر الموقع على GitHub Pages

1. أنشئي مستودع جديد على [GitHub](https://github.com)
2. ارفعي الملفات (إما `/frontend/` كاملاً أو ملف `library.html` الموحّد)
3. اذهبي لـ **Settings → Pages**
4. اختاري الفرع: **main** + المجلد: **/ (root)**
5. احفظي وانتظري دقيقة — سيظهر الرابط

## ❓ مشاكل شائعة وحلولها

**خطأ CORS عند تحميل الكتب:**
- تأكدي أن Apps Script منشور باستخدام "الجميع (Anyone)"
- بعد أي تعديل على Apps Script: اضغطي **نشر → إدارة عمليات النشر → ✏️ → النسخة الجديدة**

**الكتب لا تظهر:**
- تأكدي من رابط API في `app.js`
- افتحي Console المتصفح (F12) لرؤية الخطأ

**الاستيراد بطيء:**
- 980 كتاب تأخذ ~30-60 ثانية لأنها تُرسل على دفعات من 100
- اتركي المتصفح مفتوحاً حتى تنتهي

## ✨ المميزات

نظام مكتبة عربي متكامل متوافق مع **ملف الجرد الرسمي** للمدرسة:

### للطلاب:
- 🔍 بحث سريع باسم الكتاب أو المؤلف أو رقم التصنيف
- 📊 إحصائيات لحظية للمكتبة
- ✅ معرفة الكتب المتوفرة فوراً

### للمعلمة:
- ➕ إضافة كتاب جديد للجرد
- ✏️ تعديل بيانات أي كتاب
- 🗑️ حذف كتاب
- 📤 تسجيل إعارة (مع اسم المستعير والتاريخ)
- ↩️ تسجيل إرجاع
- 📥 **استيراد ملف Excel كامل** (980 كتاب دفعة واحدة)

### تقني:
- 💾 قاعدة بيانات مركزية في Google Sheets
- 🔄 تحديث لايف لكل التعديلات
- 🌐 يعمل من أي جهاز ومتصفح
- 📱 تصميم متجاوب للجوال والكمبيوتر

## 📁 محتوى المشروع

```
library-project/
├── frontend/
│   ├── index.html     ← الواجهة
│   ├── style.css      ← التصميم
│   └── app.js         ← المنطق
│
├── backend/
│   └── AppsScript.js  ← كود الخادم
│
├── standalone/
│   └── library.html   ← ملف موحّد (لـ Canva أو رفع سريع)
│
├── docs/
│   └── setup.md       ← دليل الإعداد الكامل
│
├── library-data.xlsx  ← الجرد الكامل (980 كتاب) للاستيراد
├── library-data.csv   ← نسخة CSV من الجرد
└── README.md
```

---

## 🚀 البدء السريع

1. أنشئي Google Sheet جديد
2. الصقي كود `backend/AppsScript.js` في Apps Script وانشريه
3. ضعي رابط الـ API في `frontend/app.js`
4. ادخلي بكلمة المرور واستوردي `library-data.xlsx`
5. ارفعي `frontend/` على GitHub Pages

📖 **الدليل الكامل**: [docs/setup.md](docs/setup.md)
